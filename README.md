♻️ EcoSortAI

EcoSortAI is an AI-powered waste segregation assistant that helps users identify and classify waste materials using Text Analysis, Image Uploads, and Camera Scanning. The application provides waste category identification, recyclability information, disposal guidance, environmental impact, and sustainability tips to promote responsible waste management.

⚙️ Installation

Clone Repository

git clone YOUR_GITHUB_REPOSITORY_LINK
cd EcoSortAI

Create Virtual Environment

python -m venv venv

Linux/Mac:

source venv/bin/activate

Windows:

venv\Scripts\activate

Install Dependencies

pip install -r requirements.txt

Configure API Key

Create a ".env" file:

GEMINI_API_KEY=YOUR_API_KEY

Run Application

streamlit run app.py

🎯 Problem Statement

Improper waste segregation is a major environmental challenge that leads to increased landfill waste, pollution, and reduced recycling efficiency. Many people are unaware of the correct methods for classifying and disposing of different types of waste. EcoSortAI addresses this issue by using Artificial Intelligence to identify waste items and provide accurate segregation and disposal recommendations.

💡 Solution

EcoSortAI uses Google's Gemini AI model to analyze waste through text descriptions, uploaded images, and camera scans. The system identifies waste categories, determines recyclability, suggests disposal methods, highlights environmental impacts, and provides sustainability tips to encourage responsible waste management.
